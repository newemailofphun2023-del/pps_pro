# Phunzaa Performance Studio PRO (PPS)

ระบบจัดการเพลงสำหรับสถานีวิทยุ
รองรับ:
- วิเคราะห์เพลง (Chord / BPM / Key)
- แยกเสียง Karaoke (Vocal / Instrumental)
- ตรวจสอบไฟล์เพลง (คุณภาพ / ซ้ำ / เสีย)
- จัดการ Library สำหรับ DJ / Broadcast
